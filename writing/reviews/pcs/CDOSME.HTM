CDoS Media Executive

Supplier: CDoS 01645 464450

Price: £2,799

Verdict: A very well-specified system, with good components, but deeply flawed by an inexperienced set-up.

Among the usual components that we would expect to see in a machine of this class, the CDoS has an interesting specification: a 28.8Kbit/s modem, tape streamer, and two sound cards.

It was odd to see two SoundBlasters listed in Windows 95’s device list. We tried removing one or the other, but it kept reappearing at the next reboot. Opening the case showed why: there are two of them. One’s a Creative Labs Vibra 16 built into the Intel Endeavour motherboard, and the other’s an AWE32 in an ISA slot. The Vibra 16 was disabled, both in the BIOS and in Windows’ device list, but it was still causing a conflict, although the AWE32 was working. Furthermore, the AWE32’s IDE CD-ROM interface was enabled, even though the machine’s quad-speed Pioneer drive was connected to the motherboard’s secondary EIDE channel, causing another resource conflict. The simplest way of dealing with this was to remove the AWE32; this resolved all these problems, but at the cost of wavetable sound. If the customer requests a particular sound card, CDoS would be better advised to use the sound-chipless version of the Endeavour motherboard. Whichever card, the speakers are Juster units, and are among the best on test.

Other devices are generally good: there’s a 4Mb Matrox Millennium, unfortunately with outdated drivers and the redundant duplicate display card in Windows’ device list, driving an Iiyama monitor. The CD-ROM and sound card have DOS device drivers installed as well as Windows ones, which is useful when you want to run software from DOS, and all the extra AWE32 Windows utilities are loaded too. A unique inclusion is a Microsoft Natural keyboard, complete with Intellitype software. These keyboards are great for touch typists and a complete pain for most other people - try before you buy!

We didn’t try the tape streamer, as there were no tapes supplied and the installed software was the Windows 3 version of Colorado Backup. (The Windows 95 version is part of the PC Pro Essentials collection on the cover CD.) You can kiss all your long filenames goodbye if you restore your backup with this version, and it might even stop the system from working properly - a serious oversight. However, with the right software it’s a superb idea - every PC user should have some backup device; floppies aren’t a viable way of backing-up modern gigabyte-plus PCs. On to the Pace internal V.34 modem: it’s set up with the correct Windows 95 driver, but on interrupt 4 - the same as COM1, which a little sticker on the back of the case helpfully tells you to is the mouse connector. With this configuration, if you move the mouse, you’ll lose the modem connection.

CDoS has tried very hard with this system, but its inexperience shows. Some of these faults could be oversights, but not all. We expect greater expertise than this, but CDoS show considerable potential.
